# sql-data-warehouse-project
Develop a modern data warehouse using PostgreSQL to consolidate raw CSV logs into a Star Schema to allow for further analysis and insight

silver.customers table has a customer_id and a customer_unique_id where there will be a unique customer_id for each order although the customer may be the same. Use customer_unique_id to determine if the person is the same person.
